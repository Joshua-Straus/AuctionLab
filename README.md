## Baseline Experiments

The project currently supports baseline comparisons between first-price and second-price sealed-bid auctions.

Each experiment compares several bidding strategies:

- Truthful bidding
- Random bidding
- Fixed bid shading at 60%, 70%, 80%, and 90% of private valuation

The simulator tracks:

- Total profit
- Average profit
- Win rate
- Average bid
- Seller revenue
- Total spending
